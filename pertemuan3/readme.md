# Analisis Script Portofolio

Saya membuat portofolio ini dengan tema bewarna pink dengan hint ungu dengan CSS bagian root agar mudah mengganti tema warna sesuai keinginan.
Pada header saya menampilkan nama saya dengan tag <h1> dan menambah deskripsi singkat dengan tag <p> sebagai subjudul.
Dibawahnya ada menu navigasi dengan menggunakan tag <nav>, yang akan menuju ke bagian tentang saya, pendidikan, dan kontak. Saya juga memberi syle berupa background pink, teks bold, dan ada hover efek naik sedikit saat kursor mengenainya.

Terdapat bagian perkenalan dengan menampilkan foto profil saya dengan style bulat dengan cara memberikan border-radius:50% juga object-fit: cover hingga foto tetap tapi dan tampil dengan lingkaran. Saya juga membuat layout dua kolom(hero-content) agar foto berada pada sisi kiri dan teks perkenalannya disamping kanan yang berisi sapaan dan deskripsi singkat lalu memberikan warna gradasi pada background agar terlihat menarik.

Pada bagian "Tentang saya" menggunakan tag <h3> juga menambahkan sedikit tentang minat saya. lalu untuk jurusan, universitas, lokasi, dan bahasa saya buat grid(about-card) dan masing-masing diberi ikon fa(fa-graduation-cap untuk ikon pada jurusan, fa-university untuk ikon pada kampus, dll)

Dalam Pendidikan dan Tujuan terdapat icon buku dan target yang menggunakan (fa-book) dan (fa-bulleye) dengan meletakkan icon pada sisi kiri dan teks pada sisi kanan menggunakan class .education-item.

Bagian kontak, saya menampilkan email dalam bentuk link (mailto) hingga bisa langsung terbuka aplikasi emailnya. Lalu terdapat ikon sosial media seperti GitHub dan instagram dengan style bulat dan efek hover yang telah terhubung langsung dengan sosial media sesuai ikonnya. Saya tambahkan tempat mengirim pesan sederhana juga sedikit javaScript agar lebih interaktif dengan menggunakan onsubmit walau belum ada servernya.


Bagian terakhir yaitu footer untuk menutup halaman dengan footer sederhana berisi copyright dengan background putih transparan agar tetap menyatu dengan temanya.
